# 🥮 Idli Roundness Checker

A fun, light-themed web tool to check how perfectly round your idli is!  
Upload an image or use your webcam to scan your idli and get instant **roundness stats**.

---

## ✨ Features
- 📷 **Dual Mode** – Upload photo or use webcam
- 🎯 **Red Circular Outline** – Detects idli shape and shows its boundaries
- 📊 **Detailed Stats** – Width, height, diameter, aspect ratio, estimated area
- ⏳ **Loading Animation** – Simulates a “calculation” scan (6–7 seconds)
- 🎨 **Light Pastel Theme** – Modern, clean, and eye-friendly
- ⚡ **Fully Offline** – Runs in your browser without a server

## 🚀 Getting Started
1. Clone this repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/idli-roundness-checker.git
   cd idli-roundness-checker
   ```
2. Open `index.html` in your browser.
3. Upload an idli photo or start the webcam.
4. Wait for the scanner animation and see your results!
